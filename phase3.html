Internship Overview

During the internship at Oktacon Softwares, the focus is on learning modern web development technologies including HTML, CSS and JavaScript. These technologies help in building responsive and interactive websites.

The training includes understanding semantic HTML structure, CSS layouts, responsive design and adding dynamic behavior using JavaScript.
